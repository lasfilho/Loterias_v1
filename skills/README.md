# Skills — Competências técnicas do projeto

Skills são **guias executáveis** com padrões, checklists e armadilhas específicas do domínio lotérico e da stack do repositório.

## Como usar

- Referencie no prompt: *"Siga `/skills/analytics-pipeline.md` ao implementar"*
- Skills complementam roles: um `data-engineer` usa `historical-ingestion.md` e `prisma-schema-design.md`
- Mantenha skills atualizadas quando mudar schema, stack ou regras de negócio

## Skills disponíveis

| `prisma-schema-design.md` | Modelagem PostgreSQL / Prisma |
