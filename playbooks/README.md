# Playbooks — Fluxos operacionais

Playbooks são **receitas passo a passo** para tarefas recorrentes no projeto. Use-os para reduzir variação entre sessões de IA e entre desenvolvedores.

## Como usar

1. Identifique o playbook adequado
2. Informe ao agente: *"Execute o playbook `ingest-historical-data.md`"*
3. Marque o checklist ao concluir cada etapa

## Playbooks disponíveis

| Arquivo | Tarefa |
|---------|--------|
| `build-new-lottery-module.md` | Adicionar nova modalidade (ex.: Lotomania) |
| `ingest-historical-data.md` | Sincronizar concursos históricos |
| `create-new-dashboard.md` | Nova tela analítica por jogo |
| `add-new-heuristic.md` | Nova estratégia de palpite |
| `validate-analytics-output.md` | QA de saídas estatísticas |
